### Bước 2 : Clone sources từ link git [text](https://github.com/DangTinh422003/quan-ly-resort-v2)
### Bước 3 : Mở cấu trúc dự án bằng phầm mềm Visual Studio code
### Bước 4 : Nhấn nút start trên thanh công cụ của phần mềm.
